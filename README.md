## Project Overview
A Game of Thrones interactive web platform hosted on a cloud server. The site allows users to explore characters, learn about the great houses of Westeros, and take a quiz to discover which house they belong to.

## Infrastructure
- **Cloud Provider:** Microsoft Azure (East Asia region)
- **VM:** Standard B2ats v2 (2 vCPUs, 1 GiB RAM)
- **OS:** Ubuntu Server 24.04 LTS
- **Web Server:** Nginx
- **Domain:** thronelore.site
- **SSL:** Let's Encrypt (HTTPS)
- **Ports open:** 22 (SSH), 80 (HTTP), 443 (HTTPS)
