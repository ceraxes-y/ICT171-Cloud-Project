Project Description

Currently, Game of Thrones fans have no single reliable place to explore the show's vast universe in an interactive way. Information is scattered across wikis, Reddit threads, and fan sites that are often outdated or overloaded with spoilers. My proposed cloud project aims to solve this by building a centralized, interactive web platform dedicated to the Game of Thrones universe.

The site will allow users to search for characters and instantly see which house they belong to, along with a detailed profile covering their role, allegiance, status, and story arc. A personality quiz will let visitors discover which Westerosi house they belong to based on their values and choices. Each of the great houses will have its own dedicated page explaining their words, seat, region, and significance in the story.

From a technical perspective, the current version is a static site built with HTML and CSS only, hosted on a Microsoft Azure cloud virtual machine running Ubuntu Server 24.04 LTS with Nginx. This initial deployment demonstrates core IaaS skills including VM provisioning, firewall configuration, and web server setup via SSH.

For the fully developed version, JavaScript will be added to handle the character search, house filtering, and quiz interactions entirely in the browser. This keeps the project lightweight and easy to serve from a basic cloud VM while still delivering a rich user experience.

The site will cover all major characters from the eight seasons of the show, including detailed information on their alliances, deaths, and key moments in the story. Users will be able to filter characters by house, status, and allegiance. The house encyclopedia will give each of the great houses its own dedicated section, covering their history, sigil, words, and most iconic members. The goal is for the site to feel like a companion you can open while watching the show. Ultimately, the aim is to create a space where any fan, whether new to Westeros or a long-time viewer, can find what they are looking for in one place.

For this project, I have chosen a CC BY-NC 4.0 or "Some Rights Reserved" license. Because this platform is dedicated to exploring the Game of Thrones universe through character profiles, house descriptions, and an interactive quiz, it is important that the content remains freely accessible to fans while preventing any commercial exploitation. Game of Thrones is an intellectual property owned by HBO, so choosing an open but non-commercial license reflects the non-profit nature of this fan project and avoids any legal conflict with the original copyright holders. Unlike a proprietary license, CC BY-NC 4.0 encourages open collaboration and knowledge sharing within educational communities. This makes it the most appropriate choice for an open, educational fan project of this nature.
